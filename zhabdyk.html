Практикалық бөлім

Серверлік жабдықтар мен желілік құрылғылар

Ұйымның IT инфрақұрылымында қолданылатын серверлер мен желілік құрылғыларға шолу.

Сервер түрлері

Rack Server

Деректер орталықтарына арналған, 19 дюймдік сөрелерге орнатылатын серверлер.

Tower Server

Шағын кеңселер мен кәсіпорындарға арналған мұнаралық серверлер.

Blade Server

Жоғары тығыздықты, модульдік серверлік шешімдер.

Желілік құрылғылар

Switch

LAN желісіндегі құрылғыларды байланыстыратын коммутатор.

Router

Әртүрлі желілер арасында деректерді бағыттайтын маршрутизатор.

Firewall

Желіні кибершабуылдардан қорғайтын қауіпсіздік құрылғысы.
