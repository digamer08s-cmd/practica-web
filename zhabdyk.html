IT Инфрақұрылым

Серверлік жабдықтар мен желілік құрылғылар

Ұйымның ақпараттық инфрақұрылымына арналған техникалық шолу. Rack, Tower, Blade серверлері және желілік коммутация негіздері.

storage

Сервер түрлері (Server Types)

Rack Server (Сөрелік сервер)

Кең таралған

Стандартты 19 дюймдік сөрелерге (rack) орнатуға арналған ықшам, жоғары өнімді серверлер. Деректер орталықтары үшін өте қолайлы.

  • check_circle Кеңістікті үнемдейді (U өлшемімен өлшенеді)
  • check_circle Кабельдерді басқару ыңғайлы

Tower Server (Мұнаралық сервер)

Дербес компьютер корпусына ұқсас жеке тұрған серверлер. Шағын кәсіпорындар мен кеңселерге арналған.

  • check_circle Орнату оңай, арнайы бөлме қажет емес
  • check_circle Салқындату жүйесі тиімді

Blade Server (Блейд сервер)

Жоғары қуат

Жоғары тығыздықты және ортақ қоректендіру жүйесі бар модульдік серверлер. Ірі компаниялар үшін тиімді шешім.

  • check_circle Модульдік архитектура және hot-swap мүмкіндігі
  • check_circle Энергияны үнемдеу

Желілік құрылғылар (Network Devices)

Switch (Коммутатор)

Жергілікті желідегі (LAN) құрылғыларды бір-бірімен байланыстыратын және деректер пакеттерін адресатқа жеткізетін құрылғы.

OSI L2/L3

Router (Маршрутизатор)

Әртүрлі компьютерлік желілер арасында деректер пакеттерін бағыттайтын құрылғы. Интернетке қосылу үшін негізгі компонент.

OSI L3

Firewall (Желіаралық қалқан)

Кіріс және шығыс желілік трафикті бақылайтын және сүзгілейтін қауіпсіздік құрылғысы. Кибершабуылдардан қорғайды.
