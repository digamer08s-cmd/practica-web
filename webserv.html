Веб-серверлер: Apache, Nginx және IIS

Ұйымның ақпараттық инфрақұрылымындағы веб-серверлердің рөлі, жұмыс істеу принциптері және олардың тиімділігін арттыру жолдары. Бұл бөлімде негізгі технологиялық шешімдер қарастырылады.

Танымал Веб-серверлерге шолу

Қазіргі таңда өндірісте кеңінен қолданылатын үш негізгі серверлік шешімнің қысқаша сипаттамасы.

flight

Apache HTTP Server

Модульдік архитектурасы бар, икемді және әлемдегі ең көп қолданылатын сенімді веб-сервер. .htaccess арқылы баптау мүмкіндігі мол.

Nginx

Жоғары жүктемеге төзімділікті (High Concurrency) және статикалық контентті жылдам өңдеуді қамтамасыз етеді. Reverse proxy ретінде танымал.

Microsoft IIS
